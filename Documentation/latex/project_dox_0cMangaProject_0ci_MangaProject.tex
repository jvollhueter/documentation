\doxysection*{Manga Project configuration.}

In this section, there are three different types of tag specified\+:
\begin{DoxyItemize}
\item \mbox{[}input\mbox{]} Required input specification embracing the next child level
\item \mbox{[}case\mbox{]} The containing specifications need to be done on the sibling level. If no child level is defined, no further specifications have to be made, if this case is choosen.
\item \mbox{[}tag\mbox{]} Case selector tag
\item \mbox{[}spec\mbox{]} Simple tag with one value only
\end{DoxyItemize}
