Methods that subclasses may redefine. 

But don\textquotesingle{}t call these methods! They are private to \mbox{\hyperlink{classnumpy_1_1distutils_1_1fcompiler_1_1FCompiler}{FCompiler}} class and may return unexpected results if used elsewhere. So, you have been warned..

\begin{DoxyVerb}Go through the self.executables dictionary, and attempt to
find and assign appropriate executables.

Executable names are looked for in the environment (environment
variables, the distutils.cfg, and command line), the 0th-element of
the command list, and the self.possible_executables list.

Also, if the 0th element is "<F77>" or "<F90>", the Fortran 77
or the Fortran 90 compiler executable is used, unless overridden
by an environment setting.

Subclasses should call this if overridden.
\end{DoxyVerb}
